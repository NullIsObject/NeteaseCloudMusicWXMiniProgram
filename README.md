## 说明

### 这是一个仿照网易云音乐听歌小程序制作的app，兼容H5端和微信小程序端，其中部分内容做了修改

## 运行

### 1.须使用下载对应的后台项目：https://github.com/Binaryify/NeteaseCloudMusicApi

### 2.运行此项目前，请先将文件："APP/src/api/request.js" 中第二行"baseUrl"，的url进行修改
